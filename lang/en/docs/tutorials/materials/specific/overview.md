# Specific Materials Examples

This document provides a comprehensive catalog of materials science tutorials organized by the MCODE (Materials Classification and Organization for Digital Exploration) system. Each entry includes:

- **Structure Notebook**: Creates the atomic structure from published data
- **Property Notebook**: Calculates and reproduces key properties from the manuscript
- **DOI Reference**: Link to the original scientific publication
